Abstract:

Criação de diversos ficheiros view em pug para estruturar a página web.

Criação de um servidor web em NodeJS que permite visualizar os Prémios, as Categorias e os Laureados.

Uso de uma base de dados MongoDB para armazenar as informações dos prémios.

Deliverables:

  • Ficheiro javascript para a API do website
  • Ficheiro javascript para interface gráfica do website
  • Ficheiro javascript para a raiz do website
  • Ficheiro pug para página geral do website
  • Ficheiro pug para visualizar todas as categorias
  • Ficheiro pug para visualizar os laureados
  • Ficheiro pug para visualizar todos os prémios
  • Ficheiro pug para pesquisar prémios por ano ou categoria
  • Ficheiro pug para visualizar todas as informações sobre um prémio
  • Ficheiro pug para página de erro
  • Ficheiro pug com a estrutura base para os outros pugs
  • Ficheiro css para dar estilo ao website
  • Ficheiro javascript que define a estrutura dos prémios na base de dados
  • Ficheiro javascript para exportar as diversas operações que a API permite
