Title: Geração de Websites e Servidores Web em NodeJS End Date: Outubro 2019
Subtitle: Inserido no contexto do TPC4 Supervisor: José Carlos Ramalho

Abstract:

Criação de uma stylesheet xsl para transformar o xml de Arqueossítios do Norte de Portugal num website em que cada arqueossítio está numerado por ordem crescente de aparência no ficheiro xml.

Criação de um servidor web em NodeJS que permite acesso aos arqueossítios, introduzindo o respetivo número no url.

Deliverables:

  • Ficheiro transformação(.xsl) dos Arqueossítios num website com o load do xsl individual embutido
  • Ficheiro transformação(.xsl) individual para cada Arqueossítio carregado em cada .xml do website
  • Servidor Web em NodeJS para aceder aos arqueossítios
